UAV Construction

Construction of a quadcopter UAV for scanning and data collection, designed to be controlled by a neural network. The drone is first flown via remote control, with the long-term goal of autonomous, onboard AI-driven navigation through an integrated microcontroller.

Quadcopter Neural Networks Arduino CleanFlight 2.6.0 Flight Controller LiPo PID Tuning Simulation

Project Description

This diploma thesis covers the construction of an unmanned aerial vehicle (UAV) capable of using artificial intelligence — specifically neural network models — for navigation and decision-making. It provides a detailed account of assembling a quadcopter that is initially controlled by a remote-control system and eventually steered automatically.

The flight controller is configured through CleanFlight (v2.6.0). The final objective is to run the neural network on a microcontroller (Arduino) mounted directly on the quadcopter, enabling real-time decisions about the drone's direction during flight.

Neural Network Training

For autonomous flight, a neural network was trained to replace conventional decision-making algorithms. A simulated drone environment was built, complete with obstacles, to generate training data.

01
Build a drone simulation environment.
02
Construct obstacles within the simulation for navigation scenarios.
03
Train the neural network (Multilayer Perceptron) on the simulated data for real-time direction decisions.
