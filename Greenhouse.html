Greenhouse Automation

Greenhouse automation system using Arduino Uno R3 with temperature and light intensity monitoring and control.

Arduino Uno R3 Servo Motor DC Motor Photoresistor Temperature Sensor

Project Description

The goal of the project is to design and demonstrate a greenhouse automation system using a microcontroller. The system reads ambient light intensity and temperature, and controls a ventilation fan and a shading mechanism accordingly.

Designed and implemented a greenhouse automation system on physical Arduino Uno R3 hardware. Circuit prototyped in Tinkercad prior to physical build.

Control Logic

01
System waits in standby — LCD displays "Patiste to button gia enarxi"
02
Button press activates system — LCD confirms "Automatismos Thermokipiou ON"
03
Photoresistor reads ambient light (LUX). If LUX > 1628.36 → Red LED ON; if LUX < 1628.36 → Red LED OFF
04
Temperature sensor reads ambient temp. If T > 33°C → Green LED ON + Fan motor activates; else → Fan OFF
05
Servo shading mechanism angle calculated: angle = (−17/33) × LUX + 227 — adjusts fins based on light intensity
06
LCD displays real-time LUX and temperature values continuously
